bin: Ejecutables generados
build: Usado por make
fonts: Fuentes
images: Imágenes y sprites
include: Archivos .hpp

    common.hpp: Código común a todo el programa
    utility.hpp: Métodos de utilidad.
    config.hpp: Variables de configuración (En tiempo de compilación)

    fonts.hpp: Código que carga las fuentes
    images.hpp: Código que carga las imágenes

    label.hpp: Clase que define una etiqueta (texto)
    button.hpp: Clase que define un botón
    numericSelector.hpp: Clase que define un seleccionador numérico.
    scrollbar.hpp: Clase que define una barra de scroll.
    selector.hpp: Clase que define un seleccionador de texto.

    map.hpp: Clase que define el mapa
    tile.hpp: Clase que define una casilla del mapa.
    mazeGenerator.hpp: Clase que define un generador de mapas
    player.hpp: Clase que define a un jugador.
    AStarTray.hpp: Clase que define una trayectoria (A*)

    mouse.hpp: Clase que define un objeto de tipo ratón. Solo existe una instacia global de esta clase en todo momento.
    scene.hpp: Clase que define la escena principal. Solo existe una instancia de esta clase en todo momento.

src: Implementación de los archivos .hpp
LICENSE: Licencia MIT
Makefile: Permite construir el programa usando make
